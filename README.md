# enext-dogs-web

## Create image with Docker in root directory
```
docker build -t enext-dogs-web .
```
## Run app with Docker
```
docker run -it -p 8080:8080 enext-dogs-web
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
